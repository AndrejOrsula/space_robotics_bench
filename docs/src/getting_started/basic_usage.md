# Basic Usage

After successful [installation](./install.md), you're ready to explore Space Robotics Bench. This guide covers the essential operations to get you started.

<div class="warning">

#### Native Installation

With native installation, the `srb` CLI is only available through the Python environment of Isaac Sim. To run the command, use the following syntax:

```bash
"$ISAAC_SIM_PYTHON" -m srb ...
```

#### Docker/Apptainer Installation

```bash
srb ...
```

</div>

## List Registered Assets & Tasks

```bash
srb ls
```
