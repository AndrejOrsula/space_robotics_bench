# Workflows

The Space Robotics Bench provides several workflow options to support different development needs and learning approaches. This section covers how to integrate SRB with popular robotics and reinforcement learning frameworks.

## Available Workflows

### Reinforcement Learning (RL)

SRB integrates with popular RL frameworks through the standard Gymnasium API:

- [Stable-Baselines3 & SB3-Contrib](./sb3_sbx.md) - A set of reliable implementations of RL algorithms
- [skrl](./skrl.md) - A modular framework for reinforcement learning
- [Dreamer](./dreamer.md) - A family of model-based RL agents

For a general introduction to reinforcement learning with SRB, see the [Reinforcement Learning](./reinforcement_learning.md) page.

### Robotics Middleware

- [ROS 2](./ros2.md) - Integration with ROS 2 for controlling robots and collecting data

## Getting Started

Each workflow documentation page provides:

1. Basic usage examples
1. Advanced configurations
1. Tips for efficient development

Choose a workflow that best matches your development needs or research focus.
