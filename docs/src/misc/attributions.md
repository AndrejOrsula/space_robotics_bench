# Attributions

This document lists the attributions for external assets used in the Space Robotics Bench.

1. [Mars 2020 Sample Tube](https://nasa3d.arc.nasa.gov/detail/Mars-2020-Sample-Tube-3D-print-files) by JPL-Caltech
